# architecture-fe

a boilerplate for fe architecture

# main functionality

- npm scripts [dev,product,deploy]

- babel [ES6 ES5]

- webpack [图片、css、js等处理]

- jest [单元测试]

- vue/react框架

# file structure

|- src
|- webpack
|- |- app
|- |- components
|- |- global
|- |- |- reset.css

babel原理：将我们的代码转化成AST-描述代码结构的树状结构。
